# manage-users

----------------------
This project is built using NodeJS and AngularJS.<br>
This project provides basic CRUD operations such as add, delete, edit and remove users.<br>

Navigation bar:<br> 
- Home: home page<br>
* Create: Create user page<br>
+ View: View and search user page<br>

----------------------
